# yourhandwritingsucks.com


### How to setup

1. Get Node / Yarn
2. Clone the Repo
3. Run the stuff
4. Profit???

```bash
git clone https://github.com/trulyronak/yourhandwritingsucks
cd yourhandwritingsucks
yarn setup # installs dependencies
yarn start # runs the webserver + backend
```
